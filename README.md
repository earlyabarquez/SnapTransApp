# SnapTransApp 📸🌐

A smart camera translation app for Android that lets you translate 
text from the real world just by pointing your camera at it.

## Features
- 📷 Real-time camera text detection
- 🌍 Multi-language translation
- 📱 Clean and simple UI

## Tech Stack
- Java (Android Studio)
- ML Kit (OCR / Text Recognition)
- Google ML Kit Translation API

## Installation
1. Clone the repository
   git clone https://github.com/earlyabarquez/SnapTransApp.git
2. Open in Android Studio
3. Build and run on your device or emulator

## Requirements
- Android 7.0 (API level 24) or higher
- Internet connection for translation
